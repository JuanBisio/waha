# Cómo obtener tu API Key de Google Gemini

Google ofrece una capa gratuita generosa para usar sus modelos de IA (Gemini 1.5 Flash/Pro).

1.  **Ingresá a Google AI Studio**:
    [https://aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey)

2.  **Iniciá sesión** con tu cuenta de Google.

3.  Hacé clic en el botón azul **"Create API key"**.

4.  Si te pide seleccionar un proyecto de Google Cloud:
    - Podés elegir uno existente o "Create API key in new project".

5.  **Copiá la clave** que empieza con `AIza...`.

## En n8n

1.  Abrí el nodo **"Google Gemini Chat Model"**.
2.  En "Credential for Google Gemini API", seleccioná **"Create New"**.
3.  Pegá tu API Key.
